# Ranger - CheatSheet

| :arrow_forward: **Navigation**                | **`g`**                            |
| --------------------------------------------- | ---------------------------------- |
| Move up/out/in/down                           | `h`, `j`, `k`, `l`                 |
| Move back/forth in history                    | `H`, `L`                           |
| Move to top of file list                      | `gg`                               |
| Move to bottom of file list                   | `G`                                |
|                                               |                                    |
| :arrow_forward: **Tabs**                      |                                    |
| New Tab                                       | `g` `n`                            |
| Close Tab                                     | `g` `c`                            |
| Switch to next tab                            | `tab`                              |
| Switch to tab n *(opens tab if not existing)* | `alt + n`                          |
|                                               |                                    |
| :arrow_forward: **Bookmarks**                 |                                    |
| *(Book)*mark directory                        | `m` `<char>`                       |
| Remove (book)mark                             | `um` `<char>`                      |
| Go to (book)marked directory                  | `'` `<char>`                       |
|                                               |                                    |
| :arrow_forward: **Display**                   |                                    |
| Sort files *(different sorting criteria)*     | `o`                                |
| Toggle show hidden files                      | `zh` or `<backspace>`              |
| Filter files                                  | `zf` `<REGEX>` or `:filter <REGEX> |
| Filter files (stacked)                        | `.` `<char>`                       |
| Directory flattening (up to depth n)          | `:flat <n>`                        |
| Change linemode *(file attributes shown)*     | `M` `<char>`                       |
|                                               |                                    |
| **:arrow_forward: Selection**                 |                                    |
| Select file                                   | `space`                            |
| Select files using regular expression         | `:mark <REGEX>`                    |
| Unselect files using regular expression       | `:unmark <REGEX>`                  |
|                                               |                                    |
| **:arrow_forward: Operations**                |                                    |
| Cut                                           | `dd`                               |
| Copy ("yank")                                 | `yy`                               |
| Paste                                         | `pp`                               |
| Delete                                        | `dD`                               |
|                                               |                                    |
| Edit                                          | `E`                                |
| Open with                                     | `r`                                |
| Rename                                        | `cw`                               |
| Bulk rename                                   | `:bulkrename`                      |
|                                               |                                    |
| Yank filename (copy filename to clipboard)    | `yf` (custom )                     |
|                                               |                                    |
| :arrow_forward: **Set file access right**     |                                    |
| Set access rights (chmod)                     | `+`                                |
| Remove file mode for user (chmod)             | `-`                                |
|                                               |                                    |
| :arrow_forward: **Other**                     |                                    |
| Run command                                   | `:`                                |
