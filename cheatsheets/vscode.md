# VSCode - CheatSheet



| Editor                          |                             |
| ------------------------------- | --------------------------- |
| **Switch to** Prev./Next Editor | `CTRL + [PgUp] or [PgDown]` |
|                                 |                             |
|                                 |                             |

| Editor Groups                         |                                                 |
| ------------------------------------- | ----------------------------------------------- |
| **Split** Editor Down (new group)     | `CTRL + ^`                                      |
| **Split** Editor Left (new group)     | `CTRL + K` `CTRL + ^`                           |
| **Split** Editor Left (in group)      | `CTRL + K` `CTRL + SHIFT + ^`                   |
|                                       |                                                 |
| **Switch to** Editor Group # (open)   | `CTRL + [Number]`                               |
| **Switch to** Editor Group Left/Right | `CTRL + K` `CTRL + [LeftArrow] or [RightArrow]` |
|                                       |                                                 |
| **Move** Editor into Prev./Next Group | `CTRL + ALT + [LeftArrow] or [RightArrow]`      |
| **Move** Editor into Group #          | `SHIFT + ALT + [Number]`                        |
| **Move** Editor Group                 | `CTRL + K` `[ArrowKey]`                         |

| Panels                       |                    |
| ---------------------------- | ------------------ |
| Show Explorer / Toggle Focus | `CTRL + SHIFT + E` |
|                              |                    |
|                              |                    |
